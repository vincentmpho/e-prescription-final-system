Server side(API)
extract hms_api and open the folder/project with visual studio 2019
in the package manager console, run:
1. add migration "initial_migration"
2. update database

once complete, run the program

Client side:
1. open the hms_2 folder with vs code
2. open a new terminal and run:
yarn install

once complete, run:
yarn dev

now open: localhost:3000
